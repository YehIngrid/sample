熱門搜尋關鍵字 本機資料

目前顯示此瀏覽器的搜尋紀錄。全體用戶統計需後端新增 POST /api/analytics/search 並在搜尋頁呼叫。

用戶畫像 需後端 API
系所 / 年級分佈
後端需提供 GET /api/admin/analytics/user-demographics，回傳各系所與年級的用戶比例，可繪製圓餅圖。招商時可精準對接目標補習班（大四→公職班、法律/會計系→證照班）。
熱門上線時段（24hr）
後端需提供 GET /api/admin/analytics/peak-hours，回傳各小時的 session 數。可繪製折線圖，告知廣告主最佳限時廣告投放時段（例：深夜 23:00–01:00）。
DAU / MAU 趨勢（近 30 天）
後端需提供 GET /api/admin/analytics/dau-mau?days=30，回傳每日活躍用戶數。這是招商的基本門檻數據，向廠商證明平台「有人氣、活躍中」。
